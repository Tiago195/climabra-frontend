# ClimaGestão

CRM e sistema de agendamento para prestadores de serviço de ar-condicionado.

## Como rodar

- Comando do workflow: `bun dev` (Vite na porta 5000)
- Instalar dependências: `bun install`

## User preferences

- Idioma: pt-BR
- Mudanças incrementais (evitar refatorações grandes sem pedir)
- Gerenciador de pacotes / runner: **bun** (usar `bun dev`, `bun install`, etc., não npm)
